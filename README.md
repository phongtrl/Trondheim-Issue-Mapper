# Trondheim Warning Map

A simple static web app for Trondheim to report issues affecting cyclists and pedestrians.

## Features

- OpenStreetMap, CyclOSM, and satellite basemaps
- Add markers, lines, and polygons with descriptions
- Edit and delete shapes
- Export the map data as GeoJSON

## Run locally

1. Open `index.html` in a browser
2. Use the draw controls on the map to add reports
3. Click `Export GeoJSON` to copy the latest feature collection

## Notes

- This app uses open tile sources and open-source Leaflet components.
- Satellite imagery is provided by EOX `s2cloudless`.
